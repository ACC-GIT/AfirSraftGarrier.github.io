AS(Android studio)的几个快捷键

发表于 2015-12-15   |   分类于 Android   |   | 阅读次数

Ctrl + Q：

  • 光标移动到某个类，然后按此快捷方式能看到类的注释说明。

Ctrl + Alt + 鼠标左键

  • 光标移动到某个类，然后按此快捷方式能看到所有该类的实现（接口）或继承（类）。

Ctrl + Alt + 方向键

  • 可以往前或往后查看前一个或后一个编辑点。
